Security

Security is part of the design, not an afterthought

The first public release stays intentionally small: one landing page, minimal exposed surface area, HTTPS-only delivery, and clear separation from future admin and tenant endpoints.

  • HTTPS-only public access behind a reverse proxy
  • Strict response headers and minimal client-side complexity
  • No exposed admin routes in version 1
  • Clear separation between public, admin, and tenant paths as the platform expands
  • Portable deployment model for controlled transfer to Raspberry Pi hosting
